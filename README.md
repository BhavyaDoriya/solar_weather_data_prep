# Solar Data Preparation Pipeline

This repository contains scripts to prepare and merge
distributed PV generation data with NCDB weather data
for solar analysis and ML workflows.

## ⚙️ Setup

```bash
git clone <repo-url>
cd solar_weather_data_prep
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt

```
Download required solar data from: https://www.nrel.gov/grid/solar-power-data  
(select any state you like)
move all the data files to "root folder(solar_weather_data_prep)/raw/solar/"
then move forward with following

## ▶️ Run Order(IMPORTANT)

### ⚠️ Do not blindly run scripts.
### Please read comments inside the code before running it.
### You have to replace NREL_API_KEY,NREL_USER_FULL_NAME,NREL_USER_EMAIL in download_ncdb_weather.py.
Also by default this code is only extracting Actual_**_DPV.csv files cause they are proper choice for training the ML model for my project.
If you want to use all HA4_UPV,DA_UPV,HA4_DPV,DA_DPV files check for the comments in extract_dpv.py and build_pv_dataframe.py
### if you have read the instructions you can either do

```bash 
python scripts/run_all_scripts.py

```
### or proceed with following(Preferred):

```bash
python scripts/extract_dpv.py
python scripts/build_pv_dataframe.py
python scripts/aggregate_5min_to_hourly.py
python scripts/extract_weather_grids.py
python scripts/download_ncdb_weather.py
python scripts/clean_ncdb_weather.py
python scripts/merge_pv_weather.py

```

## 📜 Data Sources & Credits

This project relies on publicly available datasets provided by the
U.S. National Renewable Energy Laboratory (NREL).

### 1. NCDB / NSRDB Weather Data

Hourly gridded weather data (GHI, DNI, DHI, temperature, wind speed, etc.)
is sourced from the NREL National Climate Database (NCDB):

- Source: NREL NCDB (4 km, Hourly, CONUS)
- Documentation & access:
  https://developer.nrel.gov/docs/climate/ncdb/ncdb-4km-Hourly-CONUS-v1-0-0-RCP4-5-download/

### 2. Distributed Solar PV Generation Data

Solar PV system location, capacity, and generation time series data
is sourced from NREL’s solar power datasets:

- Source: NREL Solar Power Data
- Website:
  https://www.nrel.gov/grid/solar-power-data
